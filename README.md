This repository contains code for remote controller lcd.

## 📌 Characteristics

- 📟 **Lcd:** Sunton ESP32-2432S028R (240x320 TFT con touchscreen)
- 🖥️ **MCU:** ESP32
- 📚 **Framework:** ESP-IDF v5.x
- 🎨 **Graphic framework:** LVGL
- 🎛️ **Touchscreen:** XPT2046
